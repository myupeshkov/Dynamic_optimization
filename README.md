# Динамическая оптимизация в экономике и финансах

Курс посвящен изучению основных методов вариационного исчисления, оптимального управления и динамического программирования, используемых при решении экономических задач, связанных с долгосрочным планированием инвестиций, решении задач управления финансами и моделировании других экономических процессов.
